#Instructions for OSMC set up:

1. Change default password:
> passwd

2. Install git:
> sudo apt-get install git -y

3. Create a folder:
> mkdir Documents

4. Go for it:
> cd Documents

5. Clone this repo:
> git clone https://github.com/arnaubennassar/rpi_setup.git

6. move to the folder:
> cd osmc

7. execute the script:
> sudo sh main.sh
